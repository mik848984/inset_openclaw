**Anthropic взорвала рынок ИИ, представив Claude Opus 4.5** — модель, которую компания уже официально называет *«лучшей в мире для программирования, агентных систем и работы с компьютером»*.  

Это не просто очередной апдейт.  
Opus 4.5 — серьёзная заявка на передел рынка, где доминировали GPT-5.1, Gemini 3 и Grok 4.1.

---

# Что такое Opus 4.5 и чем он отличается от прошлых моделей?

Anthropic позиционирует Opus 4.5 как **революционную LLM**, которая:

- пишет сложный код,
- отлаживает системы из нескольких компонентов,
- работает как полноценный агент,
- понимает многозадачность и контекст,
- использует компьютер как человек,
- решает реальные головоломки в реальном мире (через действия).

Компания показывает это в видеороликах: Opus 4.5 открывает программы, ищет файлы, исправляет ошибки *в несколько переходов*, работает с редакторами, использует инструменты и сам выбирает стратегию.

# Главный тезис Anthropic: «Opus 4.5 просто понимает»

В пресс-релизе компания делает необычное заявление:

> *«Когда она сталкивается со сложной многосистемной ошибкой, Opus 4.5 сам находит решение. В целом, он просто **понимает**».*

По сути, Anthropic говорит о **качественном скачке в рассуждении** (reasoning):

- меньше галлюцинаций,  
- устойчивость к неоднозначности,  
- способность сравнивать компромиссы,  
- выбор архитектуры решения,  
- корректная работа с чужим кодом.

Это первый раз, когда компания настолько агрессивно сравнивает свою модель с «всеми конкурентами сразу».

# Новая память: бесконечный чат и сжатие контекста

Одно из самых важных нововведений:

### ✔ **«Бесконечный диалог»**
Даже если модель достигает предела контекста,  
она **самостоятельно сжимает** историю — *без потери смыслов и без уведомления пользователя*.

### ✔ Контекст стал «гибким»
Opus 4.5 делает:

- динамическое сжатие,
- сохранение ключевых фактов,
- автоматическую реконструкцию упрощённого контекста.

По сути, пользователю можно вести чат **неделями**, а модель не потеряет нить.

# Opus 4.5 как агент: новые инструменты от Anthropic

Вместе с моделью компания представила **три новых инструмента**, которые превращают модель в мощный программируемый агент:

### 🟣 Tool Search Tool  
Позволяет Claude автоматически использовать поисковые системы,  
не занимая токены контекста.

### 🟣 Programmatic Tool Calling  
Мощное обновление для разработчиков:  
ИИ сам вызывает нужный инструмент, API или функцию *в зависимости от задачи*.

### 🟣 Tool Use Examples  
Новый формат обучающих примеров для инструментов —  
универсальный стандарт, позволяющий агентам работать точнее.

Это большой шаг вперёд к полноценным автономным агентам, которые:

- исследуют сайты,  
- пишут код,  
- проверяют себя,  
- запускают программы,  
- исправляют ошибки,  
- взаимодействуют с системами.
